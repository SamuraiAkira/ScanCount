# ScanCount - программа для группировки товарных позиций

## Описание

**ScanCount** — это удобное приложение для обработки и группировки товарных позиций. Программа предназначена для быстрого суммирования количества товаров по заданным правилам группировки.


## Возможности

- **Ввод данных** — вставка списка товаров в формате: `название цена количество`
- **Гибкая группировка** — настраиваемые правила группировки через файл `rules.txt`
- **Автоматическое суммирование** — подсчёт общего количества по группам
- **Сохранение результатов** — автоматическое сохранение в файл `output.txt`
- **Детальный лог** — отображение всех этапов обработки и возможных ошибок

## Требования к системе

**Программа не требует установки дополнительного программного обеспечения!**

Для работы достаточно:
- Операционная система Windows
- 50 МБ свободного места на диске

## Установка и запуск

1. Скачайте файл `ScanCount.exe`
2. Поместите его в любую удобную папку
3. Дважды кликните по файлу для запуска программы

## Настройка правил группировки (опционально)

Для более точной группировки товаров вы можете создать файл `rules.txt` в той же папке, где находится программа.

### Пример файла rules.txt:

```
Кола 0.3л
Кола 0.5л
Кола
Пепси
Спрайт
# это комментарий, строка игнорируется
Фанта
```

### Правила создания:
- Каждое правило записывается с новой строки
- Регистр символов не важен (например, "КОЛА" и "кола" считаются одинаковыми)
- Строки, начинающиеся с `#`, считаются комментариями и игнорируются
- Пустые строки игнорируются
- Более длинные правила имеют приоритет при проверке

### Как это работает:
Если в файле `rules.txt` указаны правила, программа будет группировать товары по этим правилам. Например, если в списке товаров есть:
```
Кола 0.3л 100 5
Кола 0.5л 150 3
Кола 120 2
```

То с указанными правилами они будут сгруппированы в три отдельные группы: "Кола 0.3л", "Кола 0.5л" и "Кола". Без правил все три позиции попали бы в одну группу "Кола".

Если файл `rules.txt` отсутствует или пуст, группировка выполняется по первому слову названия товара.

## Формат входных данных

В поле ввода необходимо вводить данные в следующем формате:
```
Название_товара Цена Количество
```

### Пример:
```
Молоко Домик в деревне 85.50 10
Хлеб Бородинский 45.00 5
Кола 0.3л 100 5
Кола 0.5л 150 3
```

**Примечание:** цена не участвует в группировке и может быть любым числом.

## Использование программы

1. **Запустите программу** двойным кликом по `ScanCount.exe`

2. **Введите данные** в текстовое поле в одном из форматов:
   - Вставьте из буфера обмена
   - Введите вручную

3. **Нажмите кнопку «Обработать»**

4. **Просмотрите результаты:**
   - В основном окне в логе отобразится ход выполнения
   - Откроется отдельное окно с результатом группировки
   - Результат также сохранится в файл `output.txt` в папке с программой

## Формат выходных данных

Результат сохраняется в файл `output.txt` в следующем формате:
```
Результат группировки:
Кола 0.3л: 5
Кола 0.5л: 3
Кола: 2
Молоко: 10
Хлеб: 5
```

## Возможные ошибки и их решение

| Сообщение об ошибке | Причина | Решение |
|---------------------|---------|---------|
| "Файл rules.txt не найден" | Отсутствует файл с правилами | Создайте файл rules.txt или проигнорируйте (группировка будет по первому слову) |
| "Предупреждение: строка X пропущена (некорректный формат)" | Неверный формат строки | Проверьте, что строка содержит название, цену и количество, разделённые пробелами |
| "Предупреждение: строка X пропущена (количество не число)" | Вместо количества указан текст | Убедитесь, что последнее значение в строке — это число |
| "Ошибка: поле ввода пусто" | Не введены данные | Введите или вставьте данные перед нажатием кнопки "Обработать" |

## Горячие клавиши и возможности

- **Правая кнопка мыши** в поле ввода — контекстное меню (Вырезать/Копировать/Вставить)
- **Кнопка «Вставить из буфера»** — быстрая вставка данных
- **Кнопка «Очистить поле»** — очистка поля ввода
- **Кнопка «Очистить лог»** — очистка окна с логами

## Файлы программы

После работы с программой в папке могут появиться следующие файлы:

- `ScanCount.exe` — исполняемый файл программы
- `rules.txt` — файл с правилами группировки (создаётся пользователем)
- `output.txt` — файл с результатом обработки (создаётся автоматически)

## Техническая информация

Программа разработана на языке Python с использованием библиотеки tkinter и скомпилирована в исполняемый файл `.exe`. Для работы не требуется установка Python или дополнительных библиотек.
