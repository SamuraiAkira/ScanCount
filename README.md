# ScanCount

Простая утилита на Python для обработки текстового файла с данными о товарах, группировки по названиям и подсчета общего количества.

## Описание

Приложение читает файл `input.txt`, содержащий строки с информацией о товарах, обрабатывает их, группирует товары по первому слову названия и суммирует количество. Результат сохраняется в файл `output.txt` в формате CSV, а также выводится на экран. После обработки исходный файл очищается.

## Формат входных данных

Файл `input.txt` должен содержать строки в следующем формате:
`Название товара` `Цена` `Количество`

- **Название товара** может состоять из одного или нескольких слов
- **Цена** — числовое значение (может быть целым или дробным)
- **Количество** — целое число

### Требования
- Python 3.x
- Стандартные модули: os, re, csv

----

### Установка

1. Скачайте релиз или клонируйте репозиторий:

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Убедитесь, что Python установлен:

```
python --version
```
### Использование

- Откройте файл input.txt в той же директории, что и скрипт, и заполните его данными в указанном формате.
- Запустите `cmd` и выполните команду:
```
cd c:\"Путь до файла"\ScanCount
```
- Запустите скрипт:
```
python main.py
```
Результат будет выведен в консоль и сохранен в файл output.txt.
Исходный файл input.txt будет автоматически очищен.

### Пример работы

Входной файл input.txt:

```
Молоко 75 10
Молоко топленое 85 5
Хлеб 50 3
Масло сливочное 120 2
Молоко козье 150 4
```

Вывод в консоль:

```
Результат группировки:
Молоко: 19
Хлеб: 3
Масло: 2

Запись сохранена в output.txt

Файл input очищен

Операция окончена
```

### Обработка ошибок

Скрипт обрабатывает следующие ошибки:
- Отсутствие файла input.txt.
- Некорректный формат строк (пропускает их с предупреждением).
- Ошибки преобразования количества в число.
- Прочие исключения с выводом информативного сообщения.
