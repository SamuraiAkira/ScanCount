# ScanCount — программа для группировки товарных позиций

## Описание

**ScanCount** — это современное десктопное приложение для обработки и группировки товарных позиций. Программа предназначена для быстрого суммирования количества товаров по заданным правилам группировки с интуитивным интерфейсом в стиле современных редакторов кода.

## Возможности

- **Современный интерфейс** — минималистичный дизайн с тёмной и светлой темами
- **Ввод данных** — вставка списка товаров в формате: `название цена количество`
- **Гибкая группировка** — настраиваемые правила группировки через файл `rules.txt`
- **Автоматическое суммирование** — подсчёт общего количества по группам
- **Сохранение результатов** — автоматическое сохранение в файл `output.txt`
- **Детальный лог** — отображение всех этапов обработки и возможных ошибок
- **Горячие клавиши** — поддержка стандартных сочетаний клавиш браузера
- **Переключение тем** — выбор между тёмной и светлой темой оформления

## Требования к системе

**Программа не требует установки дополнительного программного обеспечения!**

Для работы достаточно:
- Операционная система Windows 10/11
- 20 МБ свободного места на диске
- WebView2 Runtime (обычно уже установлен в системе)

## Установка и запуск

1. Скачайте [последний релиз](https://github.com/SamuraiAkira/ScanCount/releases/latest)
2. Распакуйте архив в любую папку
3. Запустите `ScanCount.exe`

## Настройка правил группировки

Для более точной группировки товаров вы можете отредактировать файл `rules.txt`, который должен находиться в одной папке с программой.

### Пример файла rules.txt:

```
Кола 0.3л
Кола 0.5л
Кола
Пепси
Спрайт
# это комментарий, строка игнорируется
Фанта
```

### Правила создания:
- Каждое правило записывается с новой строки
- Регистр символов не важен (например, "КОЛА" и "кола" считаются одинаковыми)
- Строки, начинающиеся с `#`, считаются комментариями и игнорируются
- Пустые строки игнорируются
- Более длинные правила имеют приоритет при проверке

### Как это работает:
Если в файле `rules.txt` указаны правила, программа будет группировать товары по этим правилам. Например, если в списке товаров есть:

```
Кола 0.3л 100 5
Кола 0.5л 150 3
Кола 120 2
```

То с указанными правилами они будут сгруппированы в три отдельные группы: "Кола 0.3л", "Кола 0.5л" и "Кола". Без правил все три позиции попали бы в одну группу "Кола". Если файл `rules.txt` отсутствует или пуст, группировка выполняется по первому слову названия товара.

## Формат входных данных

В поле ввода необходимо вводить данные в следующем формате:

`Название_товара Цена Количество`

### Пример:

```
Молоко Домик в деревне 85.50 10
Хлеб Бородинский 45.00 5
Кола 0.3л 100 5
Кола 0.5л 150 3
```

**Примечание:** цена не участвует в группировке и может быть любым числом.

## Использование программы

1. **Запустите программу** двойным кликом по `ScanCount.exe`
2. **Введите данные** в текстовое поле: нажмите кнопку **📋 Вставить** для вставки из буфера обмена или введите данные вручную
3. **Нажмите кнопку «⚙️ Обработать»**
4. **Просмотрите результаты:** в логе отобразится ход выполнения и возможные ошибки, откроется модальное окно с результатом группировки, результат автоматически сохранится в файл `output.txt`

## Формат выходных данных

Результат сохраняется в файл `output.txt` в следующем формате:

```
Результат группировки:
Кола 0.3л: 5
Кола 0.5л: 3
Кола: 2
Молоко: 10
Хлеб: 5
```

## Интерфейс программы

### Основные элементы управления:
| Элемент | Описание |
|---------|----------|
| Поле ввода | Область для ввода или вставки данных |
| 📋 Вставить | Вставка текста из буфера обмена |
| 🗑️ Очистить | Очистка поля ввода |
| ⚙️ Обработать | Запуск обработки данных |
| 🗑️ Очистить лог | Очистка окна с логами |
| 🌓 Переключатель темы | Выбор тёмной или светлой темы |

## Возможные ошибки и их решение

| Сообщение | Причина | Решение |
|-----------|---------|---------|
| Файл rules.txt не найден | Отсутствует файл с правилами | Создайте файл rules.txt или проигнорируйте (группировка будет по первому слову) |
| Предупреждение: строка X пропущена (некорректный формат) | Неверный формат строки | Проверьте, что строка содержит название, цену и количество, разделённые пробелами |
| Предупреждение: строка X пропущена (количество не число) | Вместо количества указан текст | Убедитесь, что последнее значение в строке — это число |
| Ошибка: поле ввода пусто | Не введены данные | Введите или вставьте данные перед нажатием кнопки "Обработать" |
| Не удалось вставить из буфера | Буфер обмена пуст или недоступен | Скопируйте данные в буфер обмена и повторите попытку |

## Файлы программы

После работы с программой в папке могут появиться следующие файлы:
- `ScanCount.exe` — исполняемый файл программы
- `rules.txt` — файл с правилами группировки (создаётся пользователем)
- `output.txt` — файл с результатом обработки (создаётся автоматически)

## Техническая информация

Программа разработана на Python с использованием фреймворка Eel для создания десктопного интерфейса на основе веб-технологий (HTML, CSS, JavaScript). WebView2 используется как встроенный браузерный движок. Приложение скомпилировано в автономный исполняемый файл `.exe` и не требует установки дополнительных компонентов.
